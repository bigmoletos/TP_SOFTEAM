RACCOURCIS NOTEPAD++ Ctl + N : Nouveau (respectant les paramètre par défaut) Ctl + O : Ouvrir (en partant du dossier du document qui à le focus, si aucun, le dossier par défaut) Ctl + S : Sauvegarder Ctl + Alt + S : Sauvegarder sous Ctl + ⇧ + S : Sauvegarder tout les documents qui ont été modifiés Ctl + W : Ferme le document qui à le focus Clic Molette : Ferme l'onglet cliqué. Alt + F4 : Quitter Alt + P : Imprimer Clic Droit + Molette : Permet de changer de fichier (onglet) en visualisant la liste de façon claire. Ctrl + Tab : Avance dans la liste des fichiers ouverts. Ctrl + ⇧ + Tab : Recule dans la liste des fichiers ouverts. Ctrl + + / - ou Ctrl + Molette : Permet de zoomer et dézoomer sur le texte. Ctrl + Z : Annuler Ctrl + Y : Refaire Ctrl + C ou Ctrl + Insert : Copier Ctrl + V ou ⇧ + Insert : Coller Ctrl + X ou ⇧ + Suppr : Couper Suppr. : Supprimer Ctrl + A : Sélectionner tout Tab : Sur un ligne : Insère un tabulation (s'il y a une sélection : remplace la sélection par la tabulation). Sur sélection de plusieurs lignes : Insère un tabulation au début de chaque ligne. ⇧ + Tab : Sur un ligne : Se déplace en arrière d'une anti-tabulation Sur sélection de plusieurs lignes : Supprimer une tabulation au début de chaque ligne (si possible). Ctrl + D : Dupliquer la ligne courante ou une groupe de ligne (premier exclu, dernière inclue si incomplète) Ctrl + I : Insère un tabulation Ctrl + J : Joindre plusieurs lignes en une seule (suppression des retours de chariot) Ctrl + ⇧ + ↑ / ↓ : Déplacer une seule ligne dans le document (vers le haut ou le bas). Si il y a une sélection, c'est la ligne où est poser le curseur qui sera prise. Alt + C : Édition en mode colonne (À préciser) Ctrl + ⇧ + U : Passer la sélection en MAJUSCULE. Ctrl + U : Passer la sélection en minuscule. Ctrl + Q : Commenter une ligne (suivant le langage de programmation). Ctrl + K : Dé-commenter une ligne (suivant le langage de programmation). Ctrl + ⇧ + K : Commenter ou dé-commenter une ligne (suivant le langage de programmation). Ctrl + ⇧ + Q : Commenter ou dé-commenter une sélection (suivant le langage de programmation). Ctrl + Espace : Demander la complétion de fonction (si elle n'est pas automatique) Ctrl + Entrée : Demander la complétion de mot (si elle n'est pas automatique) Ctrl + ⇧ + Entrée : (À préciser) Alt puis Clic gauche : Sélection en colonne (attention de presser d'abord sur Alt) Non indiqués : Ctrl + L : Supprime la ligne courante Ctrl + T : Inverse deux lignes (la courante avec la précédente) Ctrl + F5 : Écrit la date (ex. : 16:18 16/11/2008) Plug-ins[modifier] Function List (version 1.x)[modifier] Ctrl + Alt + ⇧ + L : Affiche ou masque la liste de fonctions. Ctrl + Alt + Z : Pour se rendre à la fonction précédente. Ctrl + Alt + Y : Pour se rendre à la fonction suivante.
